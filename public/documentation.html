The Movie API

The objective of this project/API is to build the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

API Endpoints and Brief Explanation

Get All Movies

URL: /movies

Method: GET

Query Parameters: None

Description: Returns a list of all movies in the archive.

Response Format:

[
    {
      "title": "2001: A Space Odyssey",
      "director": "Stanley Kubrick",
      "genre": "Sci-Fi",
      "description": "An imposing black structure provides a connection between the past and the future...",
      "imageUrl": "https://upload.wikimedia.org/wikipedia/en/1/11/2001_A_Space_Odyssey_%281968%29.png",
      "isFeatured": "yes"
    },
    ...
  ]

Get Movie by Title

URL: /movies/[title]

Method: GET

Query Parameters: title (String) - The title of the movie

Description: Returns details of a movie by its title.

Response Format:

{
    "title": "2001: A Space Odyssey",
    "director": "Stanley Kubrick",
    "genre": "Sci-Fi",
    "description": "An imposing black structure provides a connection between the past and the future...",
    "imageUrl": "https://upload.wikimedia.org/wikipedia/en/1/11/2001_A_Space_Odyssey_%281968%29.png",
    "isFeatured": "yes"
  }

Get Genre by Name

URL: /movies/genre/[name]

Method: GET

Query Parameters: name (String) - The name of the genre

Description: Returns details of a genre by its name.

Response Format:

{
    "name": "Sci-Fi",
    "description": "Science fiction (sometimes shortened to SF or sci-fi) is a genre of speculative fiction..."
  }

Get Director by Name

URL: /movies/directors/[name]

Method: GET

Query Parameters: name (String) - The name of the director

Description: Returns details of a director by their name.

Response Format:

{
    "name": "Stanley Kubrick",
    "bio": "Stanley Kubrick was an American film director...",
    "yearofbirth": "1928",
    "yearofdeath": "1999"
  }

Create a New User

URL: /users

Method: POST

Query Parameters: None

Request Body Format:

{
    "username": "newuser"
  }

Description: Creates a new user with the provided username.

Response Format:

{
    "username": "newuser"
  }

Update User Information

URL: /users/[username]

Method: PUT

Query Parameters: username (String) - The username of the user to update

Request Body Format: Any fields to update

Description: Updates the user's information based on the provided data.

Response Format:

"Username has been successfully updated!"

Add Movie to Favorites

URL: /users/[username]/[favmovies]

Method: POST

Query Parameters:

  • username (String) - The username of the user
  • favmovies (String) - The title of the movie to add to favorites

Description: Adds a movie to the user's list of favorite movies.

Response Format:

"Movie successfully added to Favourites list!"

Remove Movie from Favorites

URL: /users/[username]/[favmovies]

Method: DELETE

Query Parameters:

  • username (String) - The username of the user
  • favmovies (String) - The title of the movie to remove from favorites

Description: Removes a movie from the user's list of favorite movies.

Response Format:

"Movie successfully deleted from Favorites list!"

Delete a User

URL: /users/[username]

Method: DELETE

Query Parameters: username (String) - The username of the user to delete

Description: Deletes a user from the system.

Response Format:

"User successfully deleted!"
