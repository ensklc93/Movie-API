The Movie API

The objective of this project/API is to build the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

API Endpoints and Brief Explanation

Create a New User

URL: /users

Method: POST

Query Parameters: Username (String), Password (String), Email (String), Birthday (Date)

Description: Creates a new user with the provided Username, Password, E-Mail and Birthday

Request Body Format:

{
    "Username": "john_doe",
    "Password": "password123",
    "Email": "john_doe@example.com",
    "Birthday": "1990-01-01"
}

Response Format:

{
    "_id": "60c72b2f9b1e8a3f4c8b4568",
    "Username": "john_doe",
    "Password": "password123",
    "Email": "john_doe@example.com",
    "Birthday": "1990-01-01",
    "FavoriteMovies": []
}

Update User Information

URL: /users/[username]

Method: PUT

Query Parameters: username (String) - The username of the user to update

Description: Updates the user's information based on the provided data.

Request Body Format:

{
    "Username": "john_doe_updated",
    "Password": "newpassword123",
    "Email": "john_doe_updated@example.com",
    "Birthday": "1990-01-01"
}

Response Format:

{
    "_id": "60c72b2f9b1e8a3f4c8b4568",
    "Username": "john_doe_updated",
    "Password": "newpassword123",
    "Email": "john_doe_updated@example.com",
    "Birthday": "1990-01-01",
    "FavoriteMovies": []
}

Add a movie to User's favorites

URL: /users/[username]/movies/[movieID]

Method: POST

Query Parameters:

  • Username (String) - The username of the user
  • MovieID - The ID of the movie to add to favorites

Description: Adds a movie to the User's list of favorite movies.

Request Body: None

Response Format:

{
    "_id": "60c72b2f9b1e8a3f4c8b4568",
    "Username": "john_doe",
    "Password": "password123",
    "Email": "john_doe@example.com",
    "Birthday": "1990-01-01",
    "FavoriteMovies": ["60c72b2f9b1e8a3f4c8b4567"]
}

Remove a movie from User's favorites

URL: /users/[username]/movies/[movieID]

Method: DELETE

Query Parameters:

  • Username (String) - The username of the user
  • MovieID - The ID of the movie to remove from favorites

Description: Removes a movie from the User's list of favorite movies.

Response Format:

{
    "_id": "60c72b2f9b1e8a3f4c8b4568",
    "Username": "john_doe",
    "Password": "password123",
    "Email": "john_doe@example.com",
    "Birthday": "1990-01-01",
    "FavoriteMovies": []
}

Delete User by Username

URL: /users/[username]

Method: DELETE

Query Parameters: Username (String) - The username of the user to delete

Description: Deletes a user from the system.

Response Format:

{
    "message": "john_doe was deleted."
}
